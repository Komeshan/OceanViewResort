How to Use the Reservation System

  • Login: Use your assigned username and password to access the system.
  • View Reservations: Click "View Reservations" to see all current bookings.
  • Make New Reservation: Click "New Reservation" and fill out the form.
    • Ensure Check-In date is today or later.
    • Check-Out date must be after Check-In.
    • Room type determines the price automatically.
  • Edit Reservations: Use the "Edit" link in the reservation table to update booking info.
  • Delete Reservations: Use the "Delete" link to remove a reservation permanently.
  • Search/Filter: Use the search form above the table to find reservations by name, room type, or date range.
  • Logout: Click "Logout" to safely exit the system.
